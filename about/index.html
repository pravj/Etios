Ingredients
How did I cook this?

This site is built on top of Jekyll, and hosted on GitHub Pages. You are free to use this Jekyll theme, named Etios. Thank me later.

Deep down it uses the Materialize framework to arrange things. These beautiful icons are powered by ionicons.

If you are browsing in a bigger device, you can see banner images for all those sections in the navigation bar, they are sponsored by Pexels under CC0 License.

Once you have all that, you just need to mix them up and fry in a pan for 20 minutes. That's it.

If you are still uncomfortable with it, check out my food blog for this particular dish. A little paint over my webspace. (WIP)
